# ACL Submission Checklist

> Based on ACL 2025/2026 submission requirements.
> Used by S8_delivery.md for pre-submission verification.

## Mandatory Items

### 1. Limitations Section (**Required**)
- [ ] **CRITICAL**: Paper includes a dedicated "Limitations" section.
- [ ] **Substantive**: Limitations are specific and substantive, not token gestures.
- [ ] **Honest**: Acknowledges genuine weaknesses in approach or evaluation.
- [ ] **Scope**: Covers both methodological and empirical limitations.

> **WARNING**: ACL requires a Limitations section. Missing or insufficient limitations may affect reviews.

### 2. Ethics Statement
- [ ] **Present**: Ethics considerations are discussed.
- [ ] **Data**: Sources and licensing of all datasets documented.
- [ ] **Bias**: Potential biases in data, method, and evaluation discussed.
- [ ] **Fairness**: If applicable, fairness across demographic groups evaluated.
- [ ] **Dual use**: Potential for misuse discussed if relevant.

### 3. Responsible NLP Checklist
- [ ] **Languages**: Languages studied are explicitly listed.
- [ ] **Language diversity**: Limitations regarding language coverage acknowledged.
- [ ] **Human evaluation**: If used, guidelines for annotators are documented.
- [ ] **Annotation agreement**: Inter-annotator agreement metrics reported.
- [ ] **Compensation**: If crowdsourcing, compensation details provided.

### 4. Reproducibility
- [ ] **Code**: Code released or submitted as supplementary.
- [ ] **Data**: Datasets publicly available or creation process documented.
- [ ] **Model details**: Architecture, size, training procedure fully specified.
- [ ] **Hyperparameters**: All hyperparameters and search strategy documented.
- [ ] **Compute**: Hardware and total compute time reported.
- [ ] **Seeds**: Random seeds and number of runs reported.

### 5. LLM Usage Disclosure
- [ ] **Disclosed**: LLM usage in research/writing is declared.
- [ ] **InfiEpisteme note**: Automated pipeline usage disclosed.

### 6. Formatting
- [ ] **Page limit**: Long paper ≤ 8 pages, short paper ≤ 4 pages (excluding references and appendices).
- [ ] **Template**: Official ACL LaTeX template (acl2025.sty or later).
- [ ] **Anonymous**: Double-blind submission.
- [ ] **Appendix**: Supplementary material in appendix.
- [ ] **References**: ACL Anthology format preferred.

## Recommended

- [ ] **Error analysis**: Detailed analysis of failure cases.
- [ ] **Ablation study**: Component contributions evaluated.
- [ ] **Qualitative examples**: Representative input-output examples.
- [ ] **Efficiency**: Inference time and model size compared with baselines.
- [ ] **Multilingual**: If applicable, evaluation across multiple languages.
