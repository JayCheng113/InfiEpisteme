# Academic Writing Style Reference

> Adapted from [academic-writing-skills](https://github.com/bahayonghang/academic-writing-skills).

## Word Choice

### Weak → Strong Verbs

| Avoid | Use |
|-------|-----|
| make | construct, generate, produce |
| do | perform, conduct, execute |
| get | obtain, achieve, derive |
| use | employ, utilize, leverage |
| show | demonstrate, illustrate, reveal |
| find | discover, identify, observe |

### Informal → Academic

| Avoid | Use |
|-------|-----|
| a lot of | numerous, substantial, considerable |
| big | significant, substantial, major |
| very | highly, considerably, substantially |
| things | factors, elements, aspects |
| good | effective, efficient, superior |

### Redundant Expressions

| Cut | To |
|-----|-----|
| completely eliminate | eliminate |
| future prospects | prospects |
| past history | history |
| basic fundamentals | fundamentals |

## Hedging (avoid absolutes)

**Never use** in claims: "obviously", "clearly", "certainly", "always", "never", "prove" (unless mathematical proof).

**Use instead**: "suggests", "indicates", "appears to", "may be attributed to", "one possible explanation is".

## Tense Usage by Section

| Section | Tense | Example |
|---------|-------|---------|
| Abstract | Present (general) + Past (findings) | "We propose... Results showed..." |
| Introduction | Present (facts) + Present perfect (recent) | "Transformers are... Recent work has shown..." |
| Method | Past | "We trained... We used..." |
| Results | Past (findings) + Present (tables) | "Our method achieved... Table 1 shows..." |
| Discussion | Present (interpretation) + Past (results) | "This suggests... We observed..." |

## Common Preposition Errors

| Wrong | Correct |
|-------|---------|
| compare with (similar) | compare to |
| different with | different from |
| focus at | focus on |
| based in | based on |
| according with | according to |

## Common Collocation Errors

| Wrong | Correct |
|-------|---------|
| do an experiment | conduct/perform an experiment |
| make a conclusion | draw/reach a conclusion |
| rise a question | raise a question |
| solve problems | address/tackle problems |
